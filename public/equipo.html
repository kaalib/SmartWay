Nuestro Equipo

Este proyecto, creado por estudiantes de Ingeniería como trabajo final de grado, combina hardware y software para mejorar el transporte empresarial. Incorpora un dispositivo portátil de reconocimiento biométrico que envía datos a la nube y una plataforma web para mostrar rutas en tiempo real. Usa inteligencia artificial para planificar rutas rápidamente, organizando hasta 40 direcciones en 2 minutos. La IA analiza datos de Barranquilla para estimar tiempos de viaje, evitar tráfico y considerar arroyos o el clima, creando rutas que ahorran tiempo o evitan congestión. Una base de datos agiliza el proceso. Esta solución optimiza el despacho de empleados, garantiza su seguridad y ofrece a las empresas una herramienta moderna de gestión.
